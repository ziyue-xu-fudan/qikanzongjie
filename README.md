# Medical Journal Analysis Report Visualization

This is a Streamlit application that visualizes the analysis of medical journal papers.

## Local Setup

1.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the app:**
    ```bash
    streamlit run app_viz.py
    ```

## Deployment (Streamlit Community Cloud)

1.  **Push this code to GitHub.** Ensure the repository contains:
    *   `app_viz.py`
    *   `requirements.txt`
    *   `multi_journal_analysis_report.xlsx`

2.  **Log in to [Streamlit Community Cloud](https://share.streamlit.io/).**

3.  **Click "New app"** and select your GitHub repository.

4.  **Deploy!** Streamlit will automatically install the requirements and launch your app.
