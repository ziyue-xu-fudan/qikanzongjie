# 优化标签显示逻辑 (显式展示 III 期与 NA)

为了更精准地展示数据状态，我们将调整 `app_v2.py` 中的标签渲染逻辑。

## 1. Phase 标签逻辑优化
*   **III 期高亮**: 如果 `Phase` 包含 "III" 或 "3"，使用 **紫色背景** (`#e9d8fd`, `#44337a`) 进行强提醒。这有助于发现检索式的漏网之鱼。
*   **N/A 显式展示**: 不再隐藏 `N/A` 或空值，而是显示为 `⚪ N/A` (灰色背景)，明确告知用户该字段缺失或不适用。

## 2. Cancer Type 标签优化
*   **未指定/空值**: 如果 `Cancer_Type` 为空或 "Unknown" / "未指定"，显示为 `❓ 未指定癌肿` (灰色背景)，提示用户注意。
