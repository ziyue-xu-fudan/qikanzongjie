# 创建纯净版文献筛选列表 (Literature Screening List)

根据您的要求，我们将基于刚刚抓取的 746 篇非 RCT 肿瘤文献，创建一个专用于人工筛选的简洁 Excel 表格。

## 1. 核心任务
编写并运行 Python 脚本 `create_screening_list.py`，执行以下操作：

1.  **读取源数据**: 加载 `Top4_NonRCT_Cancer_Papers_2023_2026.xlsx`。
2.  **字段精简**: 只保留核心字段 `Journal_Category`, `Title`, `PubDate`, `Abstract`, `DOI`, `PMID`。
3.  **添加筛选列**:
    *   `Select?` (留空，供您打钩或填 Y/N)
    *   `Reason` (留空，供您记录为何选中/排除)
4.  **格式优化**:
    *   按 `Journal_Category` 排序。
    *   重命名列名为更友好的中文（或保持英文，视您偏好，这里保持英文通用性）。
5.  **保存输出**: 另存为 `Literature_Screening_List.xlsx`。

## 2. 预期结果
您将得到一个干净、无杂质的 Excel 文件，没有任何不需要的旧字段（如 Target Population），完全为您的人工筛选工作流定制。
