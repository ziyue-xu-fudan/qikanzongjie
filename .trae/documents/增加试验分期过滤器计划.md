# 在 App V2 中增加试验分期过滤器

为了更灵活地筛选研究类型，我们将在侧边栏增加 Phase/Design 过滤器。

## 1. UI 增加
*   **Filter by Phase / Design**: 一个多选框 (Multiselect)。
*   **选项来源**: 从 `df['Phase']` 动态提取所有唯一值（如 "I期", "II期", "N/A", "回顾性队列" 等）。

## 2. 逻辑实现
*   当用户选择特定 Phase 时，列表仅显示匹配的行。
*   配合现有的 `Status` 和 `Cancer Type` 过滤器，形成三维交叉筛选。

## 3. 执行
*   修改 `app_v2.py`。
