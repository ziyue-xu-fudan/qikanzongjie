# 生成文献筛选流程图 (Mermaid SVG)

为了直观展示我们的检索策略，我们将生成一张专业的 Mermaid 流程图。

## 1. 流程图内容设计
*   **输入**: PubMed Database (Top 4 Journals)
*   **过滤器 1 (基础)**:
    *   Time: 2023-2026
    *   Exclude: RCT / Phase 3 / Random / Blind
*   **过滤器 2 (结构化)**:
    *   NEJM: Background+Methods+Results
    *   Lancet: Methods+Findings+Interpretation
    *   JAMA: Importance+Design
    *   BMJ: Objectives+Design
*   **过滤器 3 (AI)**:
    *   DeepSeek V3 一句话总结
*   **输出**: 637 篇候选文献 -> 人工筛选 (Included/Excluded)

## 2. 技术实现
*   编写 `generate_flowchart.py`。
*   使用 Mermaid 语法描述流程。
*   由于本地可能没有 Mermaid CLI，我们将生成 HTML 文件，利用 CDN 渲染 Mermaid，或者直接生成 SVG 代码（如果工具允许）。
*   *更稳妥方案*: 生成一个包含 Mermaid 代码的 HTML 文件，您打开浏览器就能看到图，并可以右键保存 SVG/PNG。
