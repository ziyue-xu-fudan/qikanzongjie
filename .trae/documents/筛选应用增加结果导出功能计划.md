# 增加"已纳入文献"导出与预览功能

为了方便您下载和管理已筛选出的文献，我们将升级 `app_screening.py`。

## 1. 功能设计
*   **页面导航**: 在侧边栏增加 `Navigation` 单选框，切换 `Screening` (筛选模式) 和 `Included Papers` (结果模式)。
*   **结果页面 (Included Papers)**:
    *   **统计**: 显示已纳入的总数。
    *   **表格概览**: 使用 `st.dataframe` 展示 Title, Journal, PubDate, AI Summary。
    *   **导出 Excel**: `st.download_button` 下载筛选结果。
    *   **导出 DOI**: 提供一个 `st.code` 文本块，列出所有 DOI，一行一个，方便复制去下载全文。

## 2. 实施步骤
1.  修改 `app_screening.py`，引入页面路由逻辑。
2.  实现结果页面的 UI 和导出逻辑。
3.  刷新页面验证。

## 3. 预期效果
您筛选完一批后，切到 "Included Papers" 页，点一下下载按钮就能拿到最终名单。
