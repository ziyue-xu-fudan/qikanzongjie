# 深度美化筛选应用 (PubMed 风格)

我们将对 `app_screening.py` 进行全面重构，打造专业级文献阅读体验。

## 1. 核心改进：结构化摘要渲染 (`format_abstract`)
*   编写一个解析函数，自动检测大写关键词（如 `BACKGROUND`, `METHODS`, `RESULTS`, `CONCLUSIONS` 等）。
*   将这些关键词转换为 **加粗标题** 并强制换行，使摘要变成清晰的段落结构。
*   保留之前的关键词高亮功能。

## 2. UI 布局重构
*   **侧边栏 (Filters)**:
    *   **筛选进度**: 环形进度条。
    *   **年份筛选**: 增加 Slider (2023-2026)。
    *   **期刊筛选**: 使用 `st.multiselect`（多选框），更灵活。
    *   **关键词搜索**: 保持不变。
*   **主内容区 (Main)**:
    *   **Header**: 期刊名（蓝色 Badge）、日期（灰色 Badge）、PMID/DOI 链接。
    *   **Title**: 大号衬线字体，更像学术期刊。
    *   **AI Summary**: 使用 `st.success` 或自定义 CSS 框，置顶显示。
    *   **Abstract**: 使用新的结构化渲染函数。
    *   **Action Bar**: 底部固定或悬浮的操作按钮栏（Include/Exclude）。
